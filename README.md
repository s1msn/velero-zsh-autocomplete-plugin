# velero-zsh-autocomplete-plugin
Provides Velero CLI autocompletion for ZSH

To use it, copy the `velero.plugin.zsh` file to your zsh plugin directory and add `velero` to the plugins array of your zshrc file:

```bash
plugins=(
        ...
        velero
        ...
        )
```
